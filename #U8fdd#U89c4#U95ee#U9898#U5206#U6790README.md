# 违规问题分析功能

## 🎯 功能简介

违规问题分析是合规通平台的核心功能之一，专门用于深入分析App违规问题的多维度数据。

## 📍 访问路径

```
首页 → 趋势分析 → 违规问题 Tab
```

## ✨ 核心功能

### 1. 违规类型分布（TOP 15）
- 📊 饼图展示占比
- 📋 详细列表排名

### 2. 时间趋势分析
- 📈 按月统计变化
- 🔍 发现季节性规律

### 3. 部门关联分析（TOP 10）
- 🏢 识别部门关注重点
- 🔗 展示关联关系

### 4. 严重程度分析
- 🔴 高频问题（≥60%）
- 🟠 中频问题（≥30%）
- ⚪ 低频问题（<30%）

### 5. 数据筛选与导出
- 📅 年份筛选
- 🏛️ 部门筛选（1-5个）
- 💾 CSV格式导出

## 🚀 快速开始

1. 访问趋势分析页面
2. 点击"违规问题"标签
3. 选择年份和部门（可选）
4. 查看分析结果
5. 点击"导出分析数据"下载
