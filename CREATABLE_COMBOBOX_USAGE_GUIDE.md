# 可创建新选项功能 - 使用指南

## 快速开始

### 步骤1：打开案例编辑对话框
点击"新增案例"或"编辑"按钮，打开案例编辑对话框。

### 步骤2：选择或创建监管部门

#### 方式A：选择现有部门
1. 点击"监管部门"选择框
2. 输入关键词搜索（如"工信"）
3. 从列表中选择匹配的部门

#### 方式B：创建新部门
1. 点击"监管部门"选择框
2. 输入新部门名称（如"北京市网信办"）
3. 点击"新增：北京市网信办"选项
4. 系统自动创建并选中该部门
5. 看到成功提示："成功创建监管部门：北京市网信办（可在"部门与平台"模块中补充详细信息）"

### 步骤3：选择或创建应用平台

#### 方式A：选择现有平台
1. 点击"应用平台"选择框
2. 输入关键词搜索（如"应用宝"）
3. 从列表中选择匹配的平台

#### 方式B：创建新平台
1. 点击"应用平台"选择框
2. 输入新平台名称（如"鸿蒙应用市场"）
3. 点击"新增：鸿蒙应用市场"选项
4. 系统自动创建并选中该平台
5. 看到成功提示："成功创建应用平台：鸿蒙应用市场"

### 步骤4：完成案例编辑
填写其他必填字段后，点击"提交"按钮保存案例。

## 常见操作

### 1. 快速搜索
- 直接输入关键词，系统自动过滤
- 不区分大小写
- 支持部分匹配

### 2. 创建新选项
- 输入完整名称
- 点击"新增：[名称]"选项
- 等待创建完成
- 自动选中新创建的选项

### 3. 取消选择
- 点击已选中的选项
- 或点击选择框外部区域

### 4. 后续管理
- 进入"监管部门" → "部门与平台管理"
- 查看、编辑或删除创建的选项
- 补充详细信息（如部门级别、省份）

---

## 注意事项

1. **默认值**：新创建的监管部门默认为"国家级"，需要后续调整
2. **重复检查**：系统会自动检查是否存在相同名称，避免重复创建
3. **即时生效**：创建成功后立即可在当前表单中使用
4. **数据补充**：建议在"部门与平台管理"中补充完整信息
