## 🎬 功能演示

### 1️⃣ 拖拽排序

**操作步骤**:

1. **定位拖拽手柄**
   - 每个模块卡片左侧有一个 **⋮⋮** 图标
   - 这就是拖拽手柄

2. **开始拖拽**
   - 鼠标悬停到 **⋮⋮** 图标上
   - 鼠标指针会变成"抓手"形状
   - 按住鼠标左键

3. **移动模块**
   - 保持按住鼠标，上下移动
   - 被拖拽的模块会变成半透明
   - 其他模块会自动让位

4. **完成排序**
   - 移动到目标位置
   - 松开鼠标左键
   - 系统自动保存新顺序
   - 显示"排序已更新"提示

**视觉效果**:
```
拖拽前:                拖拽中:                拖拽后:
┌─────────┐           ┌─────────┐           ┌─────────┐
│ 模块 A  │           │ 模块 B  │           │ 模块 B  │
├─────────┤           ├─────────┤           ├─────────┤
│ 模块 B  │  ──────>  │ 模块 A  │  ──────>  │ 模块 A  │
├─────────┤           │ (半透明) │           ├─────────┤
│ 模块 C  │           ├─────────┤           │ 模块 C  │
└─────────┘           │ 模块 C  │           └─────────┘
                      └─────────┘
```

**键盘操作**（无障碍支持）:
- `Tab` - 切换焦点到拖拽手柄
- `Space` - 选中模块
- `↑/↓` - 上下移动
- `Space` - 确认位置

---

### 2️⃣ 一键开关显示

**操作步骤**:

1. **找到开关**
   - 每个模块卡片右侧有一个开关按钮
   - 旁边有状态徽章：
     - 🟢 **绿色"显示"** = 模块已启用
     - ⚪ **灰色"隐藏"** = 模块已禁用

2. **切换状态**
   - 点击开关按钮
   - 状态立即切换
   - 显示操作提示：
     - "已显示" - 模块已启用
     - "已隐藏" - 模块已禁用

3. **查看效果**
   - 前台页脚立即更新
   - 已隐藏的模块不会显示

**状态对比**:
```
启用状态:                      禁用状态:
┌──────────────────────┐      ┌──────────────────────┐
│ 📧 关于我们           │      │ 📧 关于我们           │
│ ✅ 显示  关于我们     │      │ ⚪ 隐藏  关于我们     │
│          [✏️]  [🟢]  │      │          [✏️]  [⚪]  │
└──────────────────────┘      └──────────────────────┘
```

### 3️⃣ 可视化编辑

**操作步骤**:

1. **打开编辑器**
   - 点击模块卡片上的 **✏️ 编辑按钮**
   - 弹出编辑对话框

2. **编辑内容**
   - 根据模块类型，显示不同的表单
   - 所有字段都是可视化的，无需编辑JSON

3. **保存修改**
   - 点击 **保存** 按钮
   - 系统验证并保存
   - 显示"保存成功"提示
   - 对话框自动关闭

4. **取消修改**
   - 点击 **取消** 按钮
   - 或点击对话框外部
   - 放弃所有修改

### 4️⃣ 实时预览

**操作步骤**:

1. **开启预览**

## 📝 各模块编辑说明

### 1. 关于我们 (about)

**字段**:
- **模块标题**: 显示在页脚的标题
- **描述**: 关于网站的简介文字
- **联系邮箱**: 联系邮箱地址

**示例**:
```
标题: 关于我们
描述: 合规通是一个信息透明、查询便捷的平台，用于展示各级网络监管部门发布的App违规监管案例和监管资讯。
邮箱: contact@heguantong.com
```

---

### 2. 快速导航 (navigation)

**字段**:
- **模块标题**: 显示在页脚的标题
- **导航链接**: 动态列表
  - 链接名称: 显示的文字
  - 路径: 内部路由路径

**操作**:
- 点击 **➕ 添加链接** 添加新链接
- 点击 **×** 删除链接
- 可以添加任意数量的链接

**示例**:
```
标题: 快速导航
链接:
  - 首页 → /
  - 案例查询 → /cases
  - 监管资讯 → /news
  - 趋势分析 → /trends
```

---

### 3. 友情链接 (friendly_links)

**字段**:
- **模块标题**: 显示在页脚的标题
- **友情链接**: 动态列表
  - 网站名称: 显示的文字
  - URL: 外部链接地址

**操作**:
- 点击 **➕ 添加链接** 添加新链接
- 点击 **×** 删除链接
- 链接会在新窗口打开

**示例**:
```
标题: 友情链接
链接:
  - 工信部 → https://www.miit.gov.cn
  - 网信办 → http://www.cac.gov.cn
```

---

### 4. 社交媒体 (social_media)

**字段**:
- **模块标题**: 显示在页脚的标题
- **社交平台**: 动态列表
  - 平台名称: 如"微信"、"微博"
  - 图标: 显示的图标文字（如"微"）
  - 链接: 社交媒体链接

**操作**:
- 点击 **➕ 添加平台** 添加新平台
- 点击 **×** 删除平台

**示例**:
```
标题: 关注我们
平台:
  - 微信 | 图标: 微 | 链接: #
  - 微博 | 图标: 博 | 链接: https://weibo.com/xxx
```

### 5. 订阅资讯 (newsletter)

**字段**:
- **模块标题**: 显示在页脚的标题
- **启用订阅功能**: 开关
- **描述文字**: 订阅说明

- **公司名称**: 版权所有者名称

## 💡 使用技巧

### 技巧1: 快速调整顺序

如果需要大幅度调整顺序：
1. 先将目标模块拖到大致位置
2. 再微调其他模块
3. 比逐个调整更高效

### 技巧2: 批量编辑

如果需要修改多个模块：
1. 先编辑并保存第一个
2. 再编辑下一个
3. 每次保存都会立即生效

### 技巧3: 使用预览验证

修改配置后：
1. 开启实时预览
2. 检查显示效果
3. 确认无误后关闭预览

### 技巧4: 临时隐藏模块

如果某个模块暂时不需要：
1. 使用开关隐藏
2. 不要删除配置
3. 需要时再开启

## ⚠️ 注意事项

### 1. 链接格式

**内部链接**（快速导航）:
- ✅ 正确: `/cases`
- ✅ 正确: `/news`
- ❌ 错误: `https://example.com/cases`

**外部链接**（友情链接）:
- ✅ 正确: `https://www.example.com`
- ✅ 正确: `http://www.example.com`
- ❌ 错误: `www.example.com`（缺少协议）

### 2. 邮箱格式

- ✅ 正确: `contact@example.com`
- ❌ 错误: `contact@example`（缺少域名）
- ❌ 错误: `contact.example.com`（缺少@）

### 3. 模块顺序

- 页脚模块从上到下显示
- 拖拽排序会立即保存
- 无法撤销排序操作（但可以重新拖拽）

### 4. 模块显示

- 隐藏的模块不会在前台显示
- 但配置数据仍然保留
- 可以随时重新启用

## 🆘 常见问题

### Q1: 拖拽不生效？

**可能原因**:
- 没有按住拖拽手柄（⋮⋮）
- 浏览器不支持拖拽功能

**解决方法**:
- 确保鼠标按在 ⋮⋮ 图标上
- 更新浏览器到最新版本
- 尝试使用Chrome或Firefox

### Q2: 保存后前台没有更新？

**可能原因**:
- 浏览器缓存

**解决方法**:
- 刷新前台页面（Ctrl+F5 或 Cmd+Shift+R）
- 清除浏览器缓存

### Q3: 编辑对话框打不开？

**可能原因**:
- 浏览器弹窗被拦截
- JavaScript错误

**解决方法**:
- 检查浏览器控制台是否有错误
- 刷新页面重试
- 联系技术支持

### Q4: 链接添加后无法删除？

**解决方法**:
- 点击链接右侧的 **×** 按钮
- 如果看不到按钮，尝试滚动对话框
- 保存后删除才会生效
