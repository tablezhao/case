# 违规问题分析功能 - 快速参考指南

## 🚀 快速开始

### 访问路径
```
首页 → 趋势分析 → 违规问题 Tab
```

### 基本操作
1. 选择年份
2. （可选）选择监管部门（最多5个）
3. 查看分析结果
4. 点击"导出分析数据"下载CSV

## 📊 四大分析模块

### 1. 违规类型分布（TOP 15）
- **饼图**：直观展示占比
- **列表**：排名、类型、次数、占比

**用途：** 识别最常见的违规问题

### 2. 时间趋势分析
- **折线图**：按月展示违规问题数量变化

**用途：** 发现季节性规律，评估监管效果

### 3. 部门关联分析（TOP 10）
- **卡片式**：每个违规问题显示关注的前6个部门

**用途：** 了解不同部门的关注重点

### 4. 严重程度分析
- **高频**：≥最大值60%（红色）
- **中频**：≥最大值30%（橙色）
- **低频**：<最大值30%（灰色）

**用途：** 快速识别需要重点关注的问题

## 🎯 典型应用场景

| 场景 | 操作步骤 | 关注重点 |
|------|----------|----------|
| **企业自查** | 查看高频问题 → 对照自身产品 | 高频问题列表 |
| **监管研究** | 切换年份对比 → 分析趋势 | 时间趋势图 |
| **行业报告** | 导出数据 → 深度分析 | 导出功能 |
| **合规培训** | 展示真实案例 → 讲解要求 | 部门关联 |
| **产品设计** | 了解常见问题 → 规避风险 | 违规类型分布 |

## 💡 数据解读技巧

### 违规类型分布
- **高占比** = 行业普遍问题 → 优先自查
- **低占比** = 新兴问题 → 关注趋势

### 时间趋势
- **上升** = 问题加重 → 提高警惕
- **下降** = 政策有效 → 正面参考
- **波动** = 季节性 → 结合实际

### 部门关联
- **多部门** = 重要且普遍 → 高度重视
- **单部门** = 专项领域 → 了解重点

### 严重程度
- **高频** = 必须立即整改
- **中频** = 需要引起重视
- **低频** = 了解具体情况

## 🔧 筛选技巧

| 目的 | 筛选方式 |
|------|----------|
| 全局分析 | 不选部门 |
| 专项分析 | 选1个部门 |
| 对比分析 | 选2-5个部门 |
| 年度对比 | 切换年份 |

## 📥 导出数据说明

### 导出内容
- 通报日期
- 应用名称
- 开发者
- 监管部门
- 应用平台
- 违规内容
- **违规关键词**（自动提取）

### 文件格式
- CSV格式
- UTF-8编码
- 支持Excel打开

## ⚠️ 注意事项

1. **统计口径**：基于关键词提取，同一案例可能包含多个违规问题
2. **时间范围**：基于通报发布日期，非违规发生时间
3. **筛选限制**：最多选择5个监管部门
4. **数据展示**：TOP 15类型、TOP 10关联、前15种低频

## 🔍 常见问题速查

**Q: 为什么数字比案例数多？**  
A: 统计的是关键词出现次数，一个案例可能包含多个违规问题。

**Q: 如何查看完整违规描述？**  
A: 鼠标悬停在截断的文本上，或导出数据查看。

**Q: 严重程度如何判定？**  
A: 基于出现频次自动计算，高频≥60%，中频≥30%。

**Q: 数据多久更新一次？**  
A: 实时从数据库获取，取决于案例录入频率。

## 📈 最佳实践

1. **定期查看**：每月查看一次，了解最新动态
2. **年度对比**：切换年份，分析变化趋势
3. **重点关注**：优先查看高频问题
4. **数据归档**：定期导出数据进行归档
5. **结合实际**：结合企业实际情况分析
