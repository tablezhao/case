# 合规通平台 - 首页数据看板功能演示

## 🎯 功能概览

本次升级重点优化了首页数据看板，实现了三大核心功能：

1. **环比统计分析** - 显示本月数据及相对上月的增减情况
2. **模块整合优化** - 合并相关信息，布局更简洁
3. **时间维度切换** - 支持月度/年度视图灵活切换

## 🎨 视觉效果说明

### 1. 环比数据展示

#### 增长趋势（绿色）
```
↑ +2 (+100.0%) 较上月
```
- 绿色文字
- 向上箭头
- 显示增加的数量和百分比

#### 下降趋势（红色）
```
↓ -3 (-30.0%) 较上月
```
- 红色文字
- 向下箭头
- 显示减少的数量和百分比

#### 持平状态（灰色）
```
- 0 (0.0%) 较上月
```
- 灰色文字
- 横线图标
- 显示无变化

### 2. 卡片样式

**本月数据卡片：**
- 大号数字显示本月数据
- 小号文字说明数据含义
- 底部显示环比变化
- 右上角显示图标

**整合卡片：**
- 上下两部分，中间分隔线
- 每部分独立显示一项数据
- 信息密度适中，易于阅读

### 3. 时间切换控件

**Tab样式：**
- 两个选项：月度视图 / 年度视图
- 选中状态：蓝色背景，白色文字
- 未选中状态：透明背景，灰色文字
- 位于图表右上角

## 💡 使用场景示例

### 场景1：查看本月监管动态

**操作步骤：**
1. 打开首页
2. 查看前两张卡片
3. 了解本月案例和应用数量
4. 查看环比数据，判断趋势

**示例数据：**
```
本月通报案例: 2
环比: ↑ +2 (+100.0%) 较上月
解读: 本月新增2个案例，相比上月（0个）增长100%
```

### 场景2：对比累计数据

**操作步骤：**
1. 查看第三张"累计统计"卡片
2. 了解历史累计数据
3. 与本月数据对比
4. 分析占比情况

**示例数据：**
```
累计案例: 3个
本月案例: 2个
占比: 2/3 = 66.7%
解读: 本月案例占累计案例的66.7%，说明大部分案例是本月新增
```

### 场景3：分析长期趋势

**操作步骤：**
1. 滚动到"通报趋势分析"图表
2. 默认查看月度视图
3. 点击"年度视图"切换
4. 对比不同时间跨度的趋势

**月度视图：**
```
2025-09: 1个案例
2025-10: 0个案例
2025-11: 0个案例
2025-12: 2个案例
趋势: 12月出现明显增长
```

**年度视图：**
```
2024: 0个案例
2025: 3个案例
趋势: 2025年开始有监管案例
```

### 场景4：了解最新动态

**操作步骤：**
1. 查看第四张"最近通报"卡片
2. 了解最新通报日期
3. 查看发布通报的部门
4. 点击案例查询查看详情

**示例数据：**
```
最新通报日期: 2025-12-04
发布部门: 国家计算机病毒应急处理中心
解读: 最近一次通报是今天，由国家级部门发布
```

## 🔍 常见问题

### Q1: 为什么本月数据是0？

**A:** 可能的原因：
1. 当月确实没有新增案例
2. 数据还未录入系统
3. 查询时间范围问题

**解决方法：**
- 检查数据库中是否有本月数据
- 确认数据录入是否及时
- 查看系统日志排查问题

### Q2: 环比百分比为什么是0%？

**A:** 当上月数据为0时，为避免除零错误，系统将百分比显示为0%。这种情况下，请关注绝对变化量。

### Q3: 如何切换时间视图？

**A:** 在"通报趋势分析"图表右上角，点击"月度视图"或"年度视图"Tab即可切换。

### Q4: 累计数据和本月数据有什么区别？

**A:** 
- 累计数据：从系统启用至今的所有数据
- 本月数据：仅统计当前自然月的数据
- 环比数据：本月与上月的对比

### Q5: 如何导出数据？

**A:** 目前系统支持在案例查询页面导出数据。未来版本将支持首页数据导出功能。
