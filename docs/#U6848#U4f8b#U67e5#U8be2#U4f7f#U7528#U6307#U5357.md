# 案例查询使用指南

## 快速开始

### 1. 浏览案例列表

进入"案例查询"页面，您会看到所有App违规监管案例的列表。

**桌面端显示：**
- 表格形式展示
- 包含：通报日期、应用名称、开发者/运营者、监管部门、应用平台、主要违规内容

**移动端显示：**
- 卡片形式展示
- 包含：应用名称、通报日期、监管部门、应用平台、违规内容摘要

### 2. 查看案例详情

**新的操作方式：**
- 直接点击案例条目的任意位置
- 无需寻找"查看详情"按钮
- 整行/整卡都可以点击

**视觉提示：**
- 鼠标悬停时，背景色会变化
- 应用名称会变为蓝色
- 鼠标指针会变成手型
- 移动端卡片会出现阴影效果

### 3. 使用搜索功能

**关键词搜索：**
- 在页面顶部的搜索框输入关键词
- 支持搜索：应用名称、开发者、监管部门、应用平台、违规内容
- 按回车键或点击搜索按钮开始搜索
- 点击"清除"按钮清空搜索

**搜索结果：**
- 显示匹配的案例数量
- 实时更新列表
- 保持分页功能

### 4. 使用筛选功能

**打开筛选面板：**
- 点击"筛选条件"按钮
- 展开筛选选项

**筛选选项：**
- **日期范围**：选择开始日期和结束日期
- **监管部门**：选择一个或多个部门
- **应用平台**：选择一个或多个平台

**应用筛选：**
- 设置好条件后点击"应用筛选"
- 列表会根据条件更新
- 点击"重置筛选"清除所有条件

### 5. 查看案例详情页

进入案例详情页后，您可以看到：

**顶部信息：**
- 应用名称（标题）
- 通报日期
- 监管部门
- 应用来源（平台）
- **查看原文链接**（新位置，在顶部）

**详细信息：**
- 开发者/运营者信息
- 主要违规内容详情

**操作功能：**
- 调节字体大小
- 分享案例
- 返回列表

### 6. 查看原文链接

**原文链接位置：**
- 位于详情页顶部
- 在"应用来源"信息旁边
- 带有外链图标的按钮

**使用方法：**
- 点击"查看原文链接"按钮
- 将在新窗口打开官方原始通报网页

### 7. 详情页其他功能

**调节字体大小：**
- 点击右上角的"放大"或"缩小"按钮
- 字体大小范围：12px - 24px
- 方便不同阅读习惯的用户

**分享案例：**
- 点击右上角的"分享"按钮
- 可选择分享到：
  - 微信（复制链接）
  - 微博
  - QQ
  - 直接复制链接

**返回列表：**
- 点击左上角的"返回"按钮
- 或点击底部的"返回列表"按钮

### 8. 翻页浏览

**分页控制：**
- 页面底部显示当前页码和总页数
- 点击"上一页"或"下一页"按钮切换页面
- 每页显示20条案例

**页面信息：**
- 顶部显示案例总数
- 底部显示当前页码

## 使用技巧

### 技巧1：快速浏览
- 在列表页，鼠标快速移动可以预览不同案例
- 悬停效果会帮助您定位当前查看的案例
- 应用名称会高亮显示

### 技巧2：高效查看
- 进入详情页后，关键信息都在顶部
- 无需滚动即可看到通报日期、部门、平台和原文链接
- 快速获取核心信息

### 技巧3：组合搜索
- 先使用关键词搜索缩小范围
- 再使用筛选条件精确定位
- 提高查找效率

### 技巧4：便捷分享
- 使用"复制链接"功能最快捷
- 可以分享到任何平台或应用
- 方便与同事或朋友分享案例

### 技巧5：舒适阅读
- 根据个人习惯调节字体大小
- 字体设置会应用到整个违规内容
- 提升阅读体验

### 技巧6：对比阅读
- 在详情页查看违规内容摘要
- 点击原文链接查看官方完整版本
- 可以对比理解案例详情

## 常见问题

**Q: 为什么没有"查看详情"按钮了？**
A: 
- 为了简化操作，现在整行/整卡都可以点击
- 直接点击案例条目的任意位置即可查看详情
- 这样操作更快捷，不需要精确点击按钮

**Q: 原文链接在哪里？**
A: 
- 进入案例详情页
- 在页面顶部，"应用来源"信息的旁边
- 带有外链图标的按钮

**Q: 如何知道哪些案例可以点击？**
A: 
- 所有案例条目都可以点击
- 鼠标悬停时会有视觉反馈
- 背景色变化和应用名称颜色变化

**Q: 移动端如何操作？**
A: 
- 直接点击卡片的任意位置
- 点击时卡片会有阴影效果
- 操作方式与桌面端一致

**Q: 表格内容显示不全怎么办？**
A: 
- 长文本会自动截断显示
- 鼠标悬停在文本上可以看到完整内容
- 点击进入详情页查看完整信息

**Q: 可以在列表页直接查看原文吗？**
A: 
- 目前需要先进入详情页
- 在详情页顶部点击"查看原文链接"
- 这样可以确保您了解案例的完整内容

**Q: 搜索和筛选可以同时使用吗？**
A: 
- 可以同时使用
- 先输入关键词搜索
- 再设置筛选条件
- 结果会同时满足搜索和筛选条件

**Q: 字体调节会保存吗？**
A: 
- 字体大小设置仅在当前页面有效
- 返回列表后重新进入会恢复默认大小
- 建议每次根据需要调节

**Q: 分享功能如何使用？**
A: 
- 点击右上角的"分享"按钮
- 选择分享平台或复制链接
- 微信分享需要先复制链接，然后在微信中粘贴

**Q: 如何快速找到特定案例？**
A: 
- 使用关键词搜索功能
- 输入应用名称、开发者或违规内容关键词
- 结合日期范围和部门筛选
- 快速定位目标案例

## 使用场景

### 场景1：日常监管查询
**需求：** 查看最新的App违规通报案例

**操作步骤：**
1. 进入案例查询页面
2. 浏览最新案例列表
3. 点击感兴趣的案例查看详情
4. 点击原文链接查看官方通报

### 场景2：特定应用查询
**需求：** 查找某个应用的违规记录

**操作步骤：**
1. 在搜索框输入应用名称
2. 按回车或点击搜索
3. 查看搜索结果
4. 点击案例查看详细违规内容

### 场景3：部门监管统计
**需求：** 查看某个监管部门的通报案例

**操作步骤：**
1. 点击"筛选条件"展开筛选面板
2. 选择目标监管部门
3. 点击"应用筛选"
4. 浏览该部门的所有通报案例

### 场景4：时间范围分析
**需求：** 查看某个时间段的违规案例

**操作步骤：**
1. 点击"筛选条件"展开筛选面板
2. 设置开始日期和结束日期
3. 点击"应用筛选"
4. 分析该时间段的违规趋势

### 场景5：平台违规对比
**需求：** 对比不同平台的违规情况

**操作步骤：**
1. 先筛选一个平台查看案例
2. 记录违规类型和数量
3. 重置筛选，选择另一个平台
4. 对比分析不同平台的违规特点
