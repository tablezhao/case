# 📢 浏览器标题配置功能上线

## 🎉 功能更新

**更新时间**：2025-12-09  
**更新模块**：网站基本信息配置

---

## ✨ 新增功能

### 🌟 浏览器标签标题配置

**功能亮点**：
- ✅ 可自定义浏览器标签页显示的标题
- ✅ 后台可视化配置，无需修改代码
- ✅ 修改后刷新页面即可生效
- ✅ 支持最多100个字符（建议60字符以内）
- ✅ 如果不填写，自动使用默认标题"合规通 Case Wiki"

**默认标题**：合规通 Case Wiki

---

## 📍 快速开始

### 1. 进入配置页面

```
登录后台 → 网站基本信息配置
```

### 2. 修改浏览器标题

```
┌─────────────────────────────────┐
│ 浏览器标签标题                   │
│ [合规通 Case Wiki        ]      │
│ 显示在浏览器标签页的标题          │
│ 建议不超过60个字符               │
│ 💡 如果不填写，将使用默认标题     │
└─────────────────────────────────┘
```

### 3. 保存并验证

```
1. 点击"保存更改"
2. 刷新浏览器页面
3. 查看浏览器标签的新标题
```

---

## 💡 使用建议

### 标题长度建议

| 长度范围 | 评价 | 说明 |
|---------|------|------|
| 10-30字符 | ✅ 最佳 | 完整显示，易于识别 |
| 30-60字符 | ⚠️ 可接受 | 可能部分显示 |
| 60+字符 | ❌ 不推荐 | 会被截断 |

### 标题示例

**✅ 推荐**：
```
- "合规通 Case Wiki"（15字符）
- "App监管案例查询"（10字符）
- "合规通 - 案例查询平台"（13字符）
```

**❌ 不推荐**：
```
- "合规通 - 信息透明、查询便捷的App违规监管案例查询平台"（太长）
- "网站"（太简单）
```

## 🔍 常见问题

### Q1：修改后什么时候生效？

**A**：修改后立即生效，刷新页面即可看到新标题。

### Q2：如果不填写会怎样？

**A**：系统会自动使用默认标题"合规通 Case Wiki"。

### Q3：浏览器标题和网站主标题有什么区别？

**A**：
- **网站主标题**：显示在网站页面内
- **浏览器标签标题**：显示在浏览器标签页

### Q4：标题显示不完整怎么办？

**A**：标题太长被截断了，建议缩短到60个字符以内。
