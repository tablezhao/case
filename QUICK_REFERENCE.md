# 可创建新选项功能 - 快速参考

## 🎯 功能概述
在案例编辑界面，用户可以直接创建新的监管部门和应用平台，无需跳转到其他页面。

---

## 🚀 快速使用

### 创建新监管部门
1. 打开案例编辑对话框
2. 点击"监管部门"选择框
3. 输入新部门名称
4. 点击"新增：[名称]"
5. ✅ 创建成功！

### 创建新应用平台
1. 打开案例编辑对话框
2. 点击"应用平台"选择框
3. 输入新平台名称
4. 点击"新增：[名称]"
5. ✅ 创建成功！

---

## 📝 注意事项

1. **默认值**：新部门默认为"国家级"
2. **重复检查**：系统自动避免重复创建
3. **即时生效**：创建后立即可用
4. **后续管理**：在"部门与平台管理"中补充信息
